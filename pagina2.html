Esto es una prueba del curso de git

git status---> muestra el estado de la carpeta contenido

git add . -->traquea todos los nuevos cambios en todos los archivos

git rm + el nombre del archivo con extencion---> saca del traqueo el archivo

git commit ----> guarda los cambios en la rama del repositorio

git log + el nombre del archivo con la extencion---> muestra el historial de cambios
